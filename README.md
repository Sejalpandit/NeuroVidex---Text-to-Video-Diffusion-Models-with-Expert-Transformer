# NeuroVidX

NeuroVidX is a Text-to-Video Diffusion Model project developed using Expert Transformers and Generative AI techniques.

## Features
- Text to Video Generation
- AI-based Video Synthesis
- React Frontend
- Deep Learning Integration
- Diffusion Model Architecture

## Technologies Used
- React.js
- Next.js
- TypeScript
- Tailwind CSS
- Python
- Deep Learning
- Diffusion Models
